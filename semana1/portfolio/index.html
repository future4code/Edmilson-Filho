Sobre mim

Sou desenvolvedor web full stack, atuo tanto no código da parte front end quanto do backend. Para mim, é importante entender as outras etapas e processos anteriores e posteriores ao seu, para que assim, você possa fazer o melhor trabalho para as outras áreas da aplicação web.

Experiência

Na Labenu, eu fui capaz de desenvolver vários projetos ricos e proveitosos para mim. Tais projetos foram feitos através das tecnologias mais populares como React, NodeJS e Typescript, além de MongoDB, MySQL, AWS: EC2, Lambda, API Gateway, Cloud Watch e S3, Jest. A Labenu me proporcionou tanto a teoria quanto a prática, ambas de qualidade. A cada semana era um trabalho diferente a ser feito e, um novo aprendizado para a minha carreira.
